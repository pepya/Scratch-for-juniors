## ✨ Feature Highlights

### 🌍 Bilingual — English & Bulgarian
Every page exists in both English and Bulgarian. A language switcher bar on every page lets parents switch instantly. Block names (Green Flag, Move Right, Forever…) are kept in English in the Bulgarian version so children learn the real ScratchJr terminology.

### 🃏 Block Flashcards
28 cards covering all ScratchJr blocks across 6 colour categories:
- 🟡 **Trigger** — Green Flag, Tap, Receive Message, Bump/Shake
- 🔵 **Motion** — Move Right/Left/Up/Down, Turn Right/Left, Go Home, Hop
- 🟣 **Looks** — Say, Grow, Shrink, Hide, Show, Reset Size
- 🟢 **Sound** — Play Sound, Record (Voice), Stop Sound, Play Note
- 🟠 **Control** — Wait, Repeat, Forever, Send Message, Go to Page
- 🔴 **End** — Stop, Go Back to Start

Each card has a CSS-drawn block shape, plain-language explanation, a real example, and a pro tip. Filter by category on screen or print and cut.

### 🎨 Project Gallery
18 illustrated project ideas — 3 per week — each with:
- Colourful scene preview drawn in CSS + emoji (no images needed)
- Week tag, type (Story / Game / Animation…), and difficulty level (Starter / Intermediate / Challenge)
- An imaginative "spark" description written to excite the child
- Block pills showing which ScratchJr blocks are needed
- 3 quick-start steps to begin immediately
- Filter bar to show only the current week's projects

### 🎨 Creative Expansions (Paper-First Tools)
Three printable planning tools designed to be used *before* opening the tablet:
- **Story Starter Cards** — 6 ready-made story prompts (setting + 2 characters + conflict) + 1 blank card
- **Character Design Sheet** — filled example + blank template with name, goal, ability, colours, emotions, and ScratchJr actions checklist
- **Storyboard Template** — 3-panel planning sheet with draw area, scene fields, dialogue, interactive block selector pills, and page-jump field

### 🏅 Achievement Badges & Diploma
- 6 weekly badges: 🎮 Explorer, 📖 Story Teller, 🔁 Loop Master, 📨 Game Maker, 🎵 Sound Engineer, 🏆 Creator
- Printable graduation diploma — type the child's name before printing, includes signature lines and seal
- Presented on Week 6 Graduation Day

### ⚡ Bonus Challenge Cards
18 challenges across 6 weeks, 3 per week at escalating difficulty:
- ⭐ Level 1 — Tricky
- ⭐⭐ Level 2 — Hard
- ⭐⭐⭐ Level 3 — Super Hard 🔥
- Week 6 legendary challenges: "Teach someone else", "The Everything Project", "Design Your Own Lesson"

### 🚀 What Comes After ScratchJr?
A full learning roadmap with:
- Visual timeline: ScratchJr → Scratch → Code.org → Python → HTML/CSS
- Per-stage: age range, what's new, why it comes next, direct links to free tools
- Quick age reference table (5–7, 7–9, 9–11, 11–13, 13+)
- 6 parent tip cards on keeping momentum after the program

## 📚 The 6-Week Program

| Week | Theme | Key Concepts | Friday Project |
|------|-------|--------------|----------------|
| 1 🐱 | Hello, ScratchJr! | Motion blocks, triggers, looks, drawing characters | Hello World! 🌍 |

| 3 🔁 | Loops & Repeats | Repeat loops, forever loops, patterns, dance | Dance Show! 🕺 |

| 6 🏆 | Grand Finale | Independent design, full build, polish, presentation | Graduation Day! 🎓 |

## 💡 Teaching Philosophy

**Ask, don't tell.** Every lesson script contains guiding questions. Children learn more by discovering than being shown. The teacher's job is to ask "What do you think will happen?" not "Here's how to do it."

**Bugs = learning.** When something breaks, celebrate it: *"Something unexpected happened — let's be detectives! What did you want to happen? What actually happened?"*

**Art first.** Encouraging kids to draw their own characters and worlds increases investment and motivation dramatically. The technical blocks become tools in service of *their* creative vision.

**Paper before tablet.** The storyboard and character design sheets are designed to be used *before* the tablet is opened. Planning on paper means better projects and fewer frustrated restarts.
