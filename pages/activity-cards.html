🐱 Week 1 — Hello, ScratchJr!
🛝
Week 1 · Day 1 · Monday
Meet the Playground!
▶ Open ScratchJr 🎨 Draw a character ⏱ 60 min
  • 1Open ScratchJr — explore together for 5 minutes, just look!
  • 2Find the Stage (big area), Blocks (left side), Characters (bottom)
  • 3Tap a character — what happens? Try tapping it lots of times!
  • 4Tap the paint brush 🎨 and draw YOUR OWN character
  • 5Give it a name and press the save button 💾
⭐
Today's Challenge
Name your character and save your first project. Show a grown-up!

Week 1 · Day 2 · Tuesday
Move It! Arrow Blocks
🔵 Blue blocks ↕↔ Directions ⏱ 60 min
  • 1Find the BLUE blocks — these make characters move!
  • 2Drag a "move right" block onto the script area
  • 3Tap it — watch your character go right!
  • 4Try adding left, up, down — chain them together
  • 5Use "go faster" (bigger number) and "go slower" (smaller number)
⭐

Press Play! Start Blocks
🟡 Yellow blocks 🚦 Triggers ⏱ 60 min
  • 1Find the YELLOW blocks — these start everything!
  • 2Drag the green flag block — this is like pressing Play!
  • 3Connect: Green Flag → Move Right → Move Up
  • 4Press the green flag on screen — it works!
  • 5Try the "tap character" trigger — what's different?

🔁 Week 3 — Loops & Repeats!
🤔
Week 3 · Day 1 · Monday
Why Repeat? The Loop Idea
🔁 Repeat block 💡 Big idea ⏱ 60 min
  • 1Physical warm-up: clap 5 times — that's a loop! 👏👏👏👏👏
  • 2Find the orange REPEAT block in ScratchJr
  • 3Put a move block INSIDE the repeat — see the wrapping shape?
  • 4Set repeat to 5 — run it — character walks 5 steps!
  • 5Compare: loop vs 5 separate move blocks. Which was easier to build?
⭐
Today's Challenge
Make a loop that runs 10 times. How far does the character travel?
I did it! ✓

Forever Loop! (Keep Going!)
♾️ Forever loop 🐦 Animation ⏱ 60 min
  • 1Find the FOREVER loop — it has no number, it never stops!
  • 2Put "move right, move left" inside — character bounces forever!
  • 3Try: grow + shrink inside forever = wings flapping!
  • 4Make the sun spin forever with a turn block inside
  • 5Stop it by pressing the red stop button — it pauses!

📨 Week 4 — Messages!

Week 4 · Day 1 · Monday
What's a Message?
📨 Messages 🎯 Send & Receive ⏱ 60 min
  • 1Real life warm-up: you shout "GO!" → friend starts running. That's a message!
  • 2Find SEND message block (yellow) and RECEIVE message block
  • 3Character A gets script: flag → SEND message
  • 4Character B gets script: RECEIVE message → move!
  • 5Press flag — A sends, B reacts! Make sure same colour message!
⭐
Today's Challenge

Chain Reactions!
🁣 Domino effect 3 characters ⏱ 60 min
  • 1Imagine falling dominoes — A hits B hits C hits D!
  • 2Character A: flag → action → SEND orange message
  • 3Character B: RECEIVE orange → action → SEND blue message
  • 4Character C: RECEIVE blue → action → SEND green message
  • 5Press play — watch the chain reaction! 😱
