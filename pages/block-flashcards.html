Trigger Blocks
Yellow · Go at the START of every script · These make things begin
▶ Green Flag
Green Flag
Starts the script when the big green flag button is pressed. This is the main "Play" button for your whole project.
💡Put this at the start of every script you want to run when you press Play.
Every script MUST start with a trigger block. No trigger = nothing happens!
👆 Tap
Tap (Character)
Starts the script when you tap that specific character on the screen with your finger.
💡Tap the dog → it barks. Tap the star → it disappears. Each character can have its own tap script!
A character can have BOTH a Green Flag script AND a Tap script — they work independently.
📩 Receive
Receive Message
Starts the script when a matching coloured message arrives. Waits silently until that message colour is sent by another character.
💡Character A sends a red message → Character B's red Receive block fires and it starts moving.
The colour of Send and Receive MUST match exactly. Different colour = nothing happens.
📳 Bump
Bump (Shake)
Starts the script when the device is shaken or bumped. A fun surprise trigger!
💡Shake the tablet → an earthquake happens on screen → characters fly in all directions!
Works best on real tablets/phones. Great for secret effects that surprise younger kids.
Motion Blocks
Blue · Move characters around the stage
→ Move Right
1
Move Right
Moves the character to the right by the number of steps shown. The number in the box controls how far — bigger number = farther.
💡Number 1 = tiny step. Number 10 = big jump across the screen.
Tap the number inside the block to change it. Chain multiple blocks to travel farther!
← Move Left
1
Move Left
Moves the character to the left. Same as Move Right but in the opposite direction.
💡Right then Left makes a character pace back and forth.
Characters keep facing the same direction when moving left — they don't flip around automatically.
↑ Move Up
1
Move Up
Moves the character up the screen. Use it to make things fly, jump, or float upwards.
💡Move Up + Move Down = a bouncing ball or a jumping character.
Combine with Move Right for diagonal movement — characters can move diagonally!
↓ Move Down
1
Move Down
Moves the character down the screen. For falling, sinking, or walking downhill.
💡Make rain fall with several small characters using Move Down in a loop.
Characters can go below the bottom edge of the screen and "disappear" — useful for exits!
↩️ Turn Right
1
Turn Right
Rotates the character clockwise. The number controls how many clockwise turns — 1 turn = a small rotation, 12 turns = a full 360° spin.
💡Turn Right 6 = half a turn. Turn Right 12 = full spin. Great for wheels and propellers!
Inside a loop, even a small turn creates a spinning animation effect.
↪️ Turn Left
1
Turn Left
Rotates the character counter-clockwise. Opposite of Turn Right.
💡Turn Right then Turn Left in a loop = a wobbling wiggle dance!
Alternating Turn Right and Turn Left creates a rocking or shaking effect — perfect for a scared character!
🏠 Go Home
Go Home
Instantly teleports the character back to where it started — its "home" position on the stage.
💡At the end of a game, Go Home resets every character to its starting position so the game can be played again.
A character's home is wherever you placed it when you added it. You can move it to set a new home.
💨 Hop
Hop
Makes the character jump forward with a little arc — like a hop! It moves up and then comes back down in one smooth motion.
💡Perfect for a frog jumping, a rabbit hopping, or a person skipping.
Hop always goes in the direction the character is facing. Rotate first to control the direction of the hop.
Looks Blocks
Purple · Change how characters look
💬 Say
Say
Shows a speech bubble with text above the character. Tap the block to type what you want the character to say.
💡Type "Hello!" and the character will show a speech bubble saying Hello! for a moment.
The bubble appears briefly. Use a Wait block after Say to give people time to read it.
📈 Grow
Grow
Makes the character bigger by one size step. Each Grow block adds one step of size.
💡Chain 5 Grow blocks in a row to make a character grow dramatically large!
Grow inside a loop creates a continuous growing animation. Combine with Shrink for a pulsing effect.
📉 Shrink
Shrink
Makes the character smaller by one size step. Opposite of Grow.
💡Shrink + Grow + Shrink + Grow in a loop = a heartbeat pulsing animation!
A character can shrink until it disappears completely — great for a "poof, it's gone!" effect.
🙈 Hide
Hide
Makes the character invisible — it disappears from the screen but is still there. It can reappear with a Show block.
💡Hide a star at the start → when it's "caught" it hides → looks like it was collected!
A hidden character's scripts still run! It can still send messages and react — you just can't see it.
👁️ Show
Show
Makes a hidden character visible again. Like magic — it appears out of thin air!
💡Start with a character hidden → make it appear with Show when a message arrives → surprise reveal!
Hide + Show in a forever loop = a flashing, blinking animation. Great for alarms and lights!
🔄 Reset Size
Reset Size
Returns the character instantly to its original size — no matter how big or small it has become.
💡After growing huge for a dramatic effect, Reset Size snaps it back to normal instantly.
Great at the start of a "play again" reset sequence to restore everything to its original state.
Sound Blocks
Green · Add sounds, music & voice
🎵 Play Sound
Play Sound
Plays a sound effect from ScratchJr's library — things like "pop!", "meow", "boing", and more. Tap the block to pick which sound.
💡Tap a character → it plays a "boing" sound → feels like a real button in an app!
The script continues while the sound plays. Use a Wait block if you want the next action to happen AFTER the sound finishes.
🎤 Record
Record (Voice)
Records your own voice or any sound using the device microphone and saves it to this block. Plays your recording when the script runs.
💡Record "3… 2… 1… Launch!" → attach to a rocket character → it narrates its own launch!
Record in a quiet room and speak clearly. Each block can hold one unique recording.
🔇 Stop Sound
Stop Sound
Stops any sound that is currently playing. Useful to cut a sound short before it finishes naturally.
💡Music playing in a loop → tap a button character → Stop Sound cuts the music suddenly for dramatic effect.
Stops ALL sounds — not just the one from the same character. Good for a reset or "game over" moment.
🎹 Play Note
Play Note
Plays a musical note. Tap the block to select the pitch. Chain different notes together to create a melody.
💡Connect 8 notes in a row to play "Happy Birthday" — each block is one note of the song.
Build a tap-instrument by giving each character a different note. Tap them in order to play a tune!
Control Blocks
Orange · Control timing, repeating & messages
⏳ Wait
1
Wait
Pauses the script for a set number of seconds before continuing to the next block. Controls the timing of your animation.
💡Say "Ready?" → Wait 2 → Say "GO!" — the pause makes the countdown feel real.
Wait 1 = 1 second pause. Experiment — different wait times create totally different moods!
🔁 Repeat
2
→ blocks inside
end repeat
Repeat
Runs the blocks inside it a set number of times, then moves on. The number controls how many times it repeats.
💡Repeat 3 → Move Right → runs Move Right three times in a row without needing three separate blocks.
Repeat 0 = skips the blocks inside entirely. Repeat 100 = runs a very long time. What does Repeat 1 do?
♾️ Forever
→ loops forever
↩ goes back up
Forever (Loop)
Runs the blocks inside it over and over — FOREVER — until the red stop button is pressed. It never ends on its own.
💡Forever → Grow → Shrink = wings flapping continuously, like a flying bird!
Nothing after a Forever block will ever run — it loops infinitely. Use Stop to end it.
📤 Send
Send Message
Sends a coloured signal to any character that has a matching Receive block. Used to make characters react to each other.
💡Character A: Green Flag → Send Red → Character B: Receive Red → Jump. A controls B!
ALL characters with the matching colour Receive block will react — you can trigger many characters with one Send!
📄 Go to Page
2
Go to Page
Instantly jumps to a different page (scene) in your project. The number tells it which page to go to.
💡End of Scene 1 → Go to Page 2. Tap a door → Go to Page 3. Build a whole story with page jumps!
Change the page number by tapping it. Page 1 is always your first page — count from left in the page tray.
End Blocks
Red · Go at the END of a script · These finish things off
🛑 Stop
Stop
Stops all scripts on ALL characters — like pressing the red stop button. Everything freezes instantly.
💡Catch the star → Send WIN message → Character B receives WIN → Stop. Game over!
Stop is permanent — nothing can restart after Stop except pressing the Green Flag again.
🔂 Go Back
Go Back to Start
Jumps back to the very beginning of the same script and runs it again — like pressing play a second time automatically.
💡Script: Move Right → Go Back to Start → the character slides right endlessly without a Forever loop!
This creates a loop without using the Repeat or Forever blocks. It keeps running until you press Stop.
