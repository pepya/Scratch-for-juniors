🚀 After ScratchJr Adventure

What Comes
After ScratchJr?

Your child just completed 6 weeks of real programming thinking. Here's exactly where to go next — a clear, age-matched path from beginner to confident coder.

🎓  For graduates of the ScratchJr Adventure program
💡

Why this roadmap exists

The most common question after Week 6 is: "We finished — now what?" Every stage below builds directly on what was learned in ScratchJr. The skills transfer. The excitement should too. This guide helps you pick the right next step without overwhelm.

🗺️ The Learning Path
Five Stages, Zero Gaps

Each stage introduces one new big idea. No skipping required — every tool genuinely builds on the previous one.

✅
✅ Completed!
ScratchJr
Ages 5–8  ·  iPad & Android  ·  Free

Visual block programming — sequencing, loops, events, messages, and sound. Your child has already mastered this.

🔵 Motion blocks 🟡 Triggers 🔁 Loops 📨 Messages 🎵 Sound 🎨 Drawing
What they can do now: Build animated stories, simple games, and music videos. They understand cause-and-effect programming, loops, and event-driven logic. That's real computer science — just with pictures instead of text.
🐱
⭐ Start here next
🚀 Recommended next step
Scratch
Ages 7–12  ·  Browser & App  ·  Free

The big brother of ScratchJr — same visual blocks but far more powerful. Variables, conditions, cloning, coordinates, and a massive community of shared projects.

📦 Variables ❓ Conditions (if/else) 🔢 Coordinates 🧬 Cloning 📡 Broadcasting 🌐 Cloud data
Why Scratch first: The blocks look almost identical to ScratchJr — the jump feels natural, not scary. Variables and conditions are the two new big ideas. Everything else is familiar territory with more power. Kids can also browse millions of projects others have built and remix them.
🐱 Open Scratch → ▶ Try a tutorial
🧩
→ Then
Code.org
Ages 8–14  ·  Browser  ·  Free

Structured puzzle-based courses that introduce real programming concepts in a game-like format. The Hour of Code is a great entry point — full courses go much deeper.

🧠 Algorithms 🔄 Functions 🔢 Parameters 🌐 Web basics 🤖 AI concepts
Why Code.org: The progression is carefully designed by educators. It bridges visual block coding toward text-based thinking. The "CS Fundamentals" course (Course D onwards) is ideal for ages 8–10. Used by 70 million+ students worldwide.
🧩 Start on Code.org → ⏱ Try Hour of Code
🐍
→ Then
Python
Ages 10+  ·  Trinket / Thonny / Replit  ·  Free

The first real text-based programming language. Python reads almost like English, which makes it the ideal first step away from blocks. One of the most widely used languages in the world.

📝 Text-based syntax 📋 Lists & dictionaries 🔧 Functions 📁 Files 🤖 Turtle graphics
Best tools to start: Trinket.io runs Python in the browser with no install. Thonny is a free desktop app designed for beginners. Replit is great for collaborative coding. Start with Turtle graphics — kids can draw and animate immediately, which feels like ScratchJr but with real code.
🐍 Try Trinket → ⬇ Download Thonny 🌐 Try Replit
🌐
→ Then
HTML & CSS
Ages 11+  ·  Any browser  ·  Free

Build real websites — the same technology this site is built with. HTML is the structure, CSS is the style. Instantly visible results make it deeply satisfying for kids who love design.

🏗️ HTML structure 🎨 CSS styling 📐 Layouts 🔤 Typography 📱 Responsive design
Why HTML/CSS is special: The result is immediately shareable with anyone in the world. A child can say "I made this website" and send a link. That pride is unmatched. Start with Khan Academy's HTML/CSS course or freeCodeCamp — both are completely free.
🌐 Khan Academy HTML → 📚 freeCodeCamp

👨‍👩‍👧 For Parents
How to Keep the Momentum
⏳
Don't rush the next stage

A child who is still excitedly building in ScratchJr after Week 6 isn't "stuck" — they're deepening mastery. Only move on when they're genuinely curious about what they can't yet do.

🎯
Follow their interest

A child who loves stories → Scratch storytelling. Loves games → Scratch game-making. Loves art → HTML/CSS. Loves logic puzzles → Python. The tool should serve their passion, not the other way around.

🐛
Bugs are the curriculum

When something breaks, resist fixing it for them. Ask "what did you expect to happen?" and "what actually happened?" The gap between those two is where real learning lives.

🌍
Share their work

A Scratch project shared with a grandparent, a website sent to a friend, a game played by a sibling — real audiences make the work feel real. Publish and share early and often.

📚
Great books to read together

Hello Ruby by Linda Liukas (ages 5–8) · Secret Coders graphic novel series (ages 8–12) · Python for Kids by Jason Briggs (ages 10+) · How to Be a Coder by DK (ages 9+)

🏫
Clubs & communities

Scratch has a huge online community at scratch.mit.edu/explore. CoderDojo runs free coding clubs in 100+ countries for ages 7–17. Code Clubs are in libraries worldwide — all free, all welcoming.
